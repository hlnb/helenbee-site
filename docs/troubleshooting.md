# Troubleshooting Guide

## Common Issues

### Build Issues

- Problem: Build fails with node version error
  - Solution: Check netlify.toml node_version setting

### Development Environment

- Problem: npm install fails
  - Solution: Clear npm cache and retry

### Content Management

- Problem: Images not optimizing
  - Solution: Check image path and format

### Deployment

- Problem: Deploy preview fails
  - Solution: Check build logs in Netlify
