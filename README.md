# Helen Bee's Website

## Grid Layout System

This site uses grid layouts inspired by and adapted from [Layout Love](https://stuffandnonsense.co.uk/layoutlove) by Andy Clarke of Stuff & Nonsense Ltd. The modular grid system provides flexible, responsive layouts perfect for blog content and portfolios.

Original

## Brand Elements

### Colors

- Primary: #332E63 (Deep purple)
- Secondary: #38A3CD (Bright blue)
- Accent Colors:
  - #F5934E (Orange)
  - #CD80B4 (Pink)
- Supporting Colors:
  - #A586BC (Light purple)
  - #873A7D (Deep pink)
  - #913073 (Another deep pink)

### Typography

- Headings (H1-H3): Stackyard font
- Body Text & Smaller Headings (H4-H6): Typewriter font
